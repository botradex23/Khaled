Binance API Endpoints

GET /api/binance/ping

Check if the Binance API service is available.

GET /api/binance/prices

Get current prices for all trading pairs.

GET /api/binance/price/<symbol>

Get the current price for a specific trading pair.

Example: /api/binance/price/BTCUSDT

GET /api/binance/ticker/24hr

Get 24-hour ticker statistics for trading pairs.

Query parameters:

  • symbol (optional): Specific trading pair (e.g., BTCUSDT)

All API responses are in JSON format.
